說明你的程式設計(功\能，流程，使用的data structure)
	功\能
	將輸入的原始程式切token(class tablefc)
		void build1to4():將輸入檔1、2、3、4table內資料放入和其檔名數字編號對應的4個vector<string>型別的 Table1、Table2、Table3和Table4，並設Table5、Table6和Table7大小為100(範圍為0~99)
		void read(string & fname):分行讀入user輸入的原始程式檔，並一一放入vector<string> instr
		void cut():將instr內原始程式分行做切token
		int sum(string str):計算並回傳傳入變數str內各字元的ASCII總和(十進位)
		void hash( string str, int tb):hash function，將sum(string str)回傳的總和mod 100後所得(為str在table內的位置)放入編號為tb的table相應位置內，若遇碰撞則存入下一個位置，再遇碰撞以此類推
		int issame(int & val, string key):檢查傳入字串key是否重複出現，是，val設為在table內所在位置，並回傳table編號，否，val設為0，並回傳0
		main():主程式
	文法檢查(class Syntax )
		vector<tkgrp> grp;內存文法檢查後的資訊，存token group
		void check(vector<tk> altk, vector<string> instr):Syntax 分析，文法檢查，vector<tk> altk內存切好的token資訊，vector<string> instr內存原始程式
	翻機器碼(class MCC)
		vector<opcode> opc;指令集，opcode表
		vector<mch> mc;內存所有指令的機器碼
		vector<confi> cft;組態表
		void bdopcode():建opcode表
		void mcc(vector<tkgrp> grp, vector<string> instr, string fname):
		mch sic(string lebaladdr, mch code, tkgrp grp):sic指令翻機器碼
		mch createmc(string ta, mch code, string format, tkgrp grp, string pc, vector<tkgrp> vgrp):sicxe指令翻機器碼
		string cvt(string pc, int cvt):將string pc轉成十進位，cvt為string pc目前進位表示
		cvttotw(string str, int cvt):將string str轉成2進位，cvt為string pc目前進位表示
		caldisp(string ta, string pc):算disp，ta-pc
		string addsix(string num, string num1):做16進位加法(string num+string num1)
		string int2str(int &i):int轉成string
		string pcadd( string tmp, int ADD):位址做加法(string tmp+ADD)
		lithash( lit str):literal hash function
		
	流程
		1.	build1to4()  
			先將資料存入1、2、3table，並設定5、6、7table size
		2.	read(string & fname)
			讀入輸入檔內原始程式
		3.	cut()        
			切token，遇到Space/Tab/Enter(white space)/Delimiter，到各table檢查是否為已設定的指令、符號等…，是則建立token，遇到Delimiter，到各table檢查是否為已設定的指令、符號等…，是則建立token(可能有一個或兩個)，查table沒此token，token為symbol、interger/Real、String之一，以mod 100 hash function放入table，遇碰撞則index+1
		4.	check(tb.altk, tb.instr)
			文法檢查
		5.	bdopcode()
			建opcode表
		6.	mcc(sy.grp, tb.instr, fname)
			翻機器碼
		7.	ofstream       
			將結果放入輸出檔，格視為一行source，一行為output出來的結果
		
	使用的data structure
		struct table:用來存7個table，table型別皆為vector<string>，分別是Table1(Instruction)、Table2(Pesudo&Etra)、Table3(Register)、Table4(Delimiter)、Table5(Symbol)、Table6(Integer/Real)、Table7(String)
		struct tk: 用來存切好的token資料，3個欄位為string name、int val和int tb，分別存切完的token名、token在所屬table的位置和所在的token編號
		class tablefc:將會用到的function做封裝，內有build1to4()、read(string & fname)、cut()、sum(string str)、hash( string str, int tb)和issame(int & val, string key)，共6個function
		vector<string> instr:用來放輸入的原始程式指令
		vector<tk> altk:用來存所有切完的token資料
		struct tkgrp:文法檢查後存tk label、tk opcode、tk op1、tk op2
		struct opcode:存opcode表資訊(string name、string format、string opcode)
		struct mch:存位址、機器碼、是否翻完機器碼(string addr、string mc、string done)
		struct confi:存組態表資訊(string lebal、vector<int> mcip、vector<string> addr、vector<string> format、vector<mch> cd)
		struct lit:存literal資訊
